Документація до програми "Skillonomy"

===== Опис

Програма "Skillonomy" розроблена для моделювання і відображення інформації про студентів, викладачів, курси та обмін токенами у вищому навчальному закладі.

===== Структура проекту

Проект має наступну структуру файлів:

- Main.scala: Головний файл програми, який об'єднує студентів, викладачів та виводить результати.
- Student.scala: Клас, що описує студента та його характеристики.
- Teacher.scala: Клас, що описує викладача та його характеристики.
- Tokenization.scala: Об'єкт, який визначає правила токенізації та розподілу токенів.
- Generator.scala: Об'єкт, який забезпечує генерацію випадкових даних для створення студентів та викладачів.
- Course.scala: Клас, що описує курс та його характеристики.
- Human.scala: Клас, що визначає загальні властивості людини, які використовуються студентами та викладачами.

===== Функціональність

Програма генерує дані для студентів та викладачів, обчислює різні статистики та виводить результати у зручному форматі. Кожен студент отримує токени за свої курси, а викладачі мають можливість взаємодіяти з цими токенами.
